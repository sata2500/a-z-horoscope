**Az-Horoscope** is a professional, AI-powered astrology platform that works with real astronomical data. It combines the ancient wisdom of astrology with the power of modern technology to provide users with personalized insights.

## ‚ú® Features

- **AI-Powered Daily Horoscopes:** Personalized daily readings generated by Google Gemini 2.5 Flash.
- **Natal Chart Analysis:** Professional birth chart calculations using the Swiss Ephemeris.
- **Personal Journal:** Keep a diary, track your mood, and discover correlations with planetary transits.
- **Weekly/Monthly Forecasts:** Plan ahead with weekly and monthly horoscope readings.
- **Email Notifications:** Receive your daily horoscope directly in your inbox.
- **Admin Panel:** Manage users and view system statistics.
- **Modern UI/UX:** Responsive design with dark/light mode support.

## üöÄ Getting Started

### Prerequisites

- Node.js v22.x
- npm v10.x
- PostgreSQL database
- Google Cloud Project (for OAuth and Gemini API)

### 1. Clone the Repository

```bash
git clone https://github.com/sata2500/a-z-horoscope.git
cd a-z-horoscope
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Set Up Environment Variables

Create a `.env` file in the root of the project and add the following variables:

```env
DATABASE_URL="..."
GOOGLE_API_KEY="..."
GOOGLE_CLIENT_ID="..."
GOOGLE_CLIENT_SECRET="..."
AUTH_SECRET="..."
NEXTAUTH_URL="http://localhost:3000"
RESEND_API_KEY="..."
```

### 4. Run Database Migrations

```bash
npm run prisma migrate dev
```

### 5. Start the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.
